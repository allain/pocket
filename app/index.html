Getting started

Since you can see this, you should have a basic template for a Hoodie app in place which already takes care of importing hoodie.js and initializing it.

The index.html includes all the code you need to sign up, sign in and write some data to hoodie. Try it out and simultaneously inspect your data in Pocket, it should update on the fly.
